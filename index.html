Programming weird things

Hi! I'm ./lemon (a.k.a ./lemon.sh). I'm 14 years old and I was born in Poland. I love programming. I code in C, C++, Java, Rust, C# and x86 (32-bit) assembly. Sometimes people say that I'm very talented for my age, but I think I just got good at programming as I spent a lot of time doing it.

I hate dynamically typed languages like JavaScript or Python, because I want the language to be strict (that's also why I love Rust). My code has to be fast, to the point, where I sometimes sacrifice code readability to gain more performance (which is generally a bad idea).

Here, I'll be posting my project ideas, thoughts, discoveries etc. Hope you'll like it!
