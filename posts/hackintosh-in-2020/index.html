Hackintoshing in 2020, is it worth it?

./lemon published on November 21, 2020

3 min, 532 words

Tags: operating-systems hackintosh

When using Windows, usually after about 8 months after installing the OS I feel like my computer is getting more and more bloated with apps I forgot to remove and stuff on the system drive I forgot to delete, so it's quite natural for me to do a system reinstall once in a while. I do it usually when a new Windows 10 build is released so I can get a clean and up to date desktop.

This time though, I've decided to play around with... macOS instead of installing a fresh copy of Windows immidiately.

I don't own a Mac (yet), but I do know how to make a Hackintosh (a non-apple machine that runs macOS). Unfortunately, I use a GTX 970 GPU and macOS doesn't support modern Nvidia cards natively. This problem can be solved using NVIDIA Web Drivers but then again, these are available only up to High Sierra (10.13). Despite this limitation, I've decided to give it a try.

To create the Hackintosh I used the OpenCore bootloader. I won't cover the details of setting up a Hackintosh, but this guide is a very good source.

The Hackintosh was a success - I got everything working, except iServices (iMessage, FaceTime etc.), because I was too lazy to set them up (some tweaks are required to get iServices working on Hackintoshes) and also I don't really use iServices so I didn't care.

I have to admit - macOS is really a beautiful experience. I love the look of GUIs, and also software development is - unlike Windows - very smooth. The desktop experience is just wonderful! I've never used an operating system like this before. Even though it's a Hackintosh, it runs way better than Windows on this machine. It even made me think about actually buying a Macbook, but they are very expensive, and I'll have to save some money before I can buy one.

Anyway, regarding the older version of macOS - at the time of writing, High Sierra is actually still supported. App Store is kinda useless though, because most apps available there require a newer macOS version. However, I was able to download apps I wanted from their official websites (for example the Facebook Messenger or Microsoft Office) in the latest version, except Adobe Creative Cloud, where I had to install a slightly older version (CC2020 instead of CC2021).

Because I have two SSDs installed in my PC, I also created a dualboot with Windows. The OpenCore bootloader allows you to select the OS you want to boot after you turn on the computer (much like GRUB).

We haven't answered one important question though... is it worth it? Well, Apple is moving to their Apple Silicon ARM chips, which will pretty much kill Hackintoshing in its current form. Also Apple's price-to-performance ratio is getting better (thanks to their own architecture) and buying a Mac starts to actually be a reasonable option. On the other hand, Apple won't cut support for x86 Macs yet, so new Hackintoshes should work for at least five years, so if you really want to build a Hackintosh, go for it!

And of course... thanks for reading!
